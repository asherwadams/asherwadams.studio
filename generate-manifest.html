How to use this tool

  1. Rename your photos: 1japan.jpg 2hk.jpg 3china.jpg etc.
  2. Drop them onto this page (or click to select).
  3. Click Generate manifest.json.
  4. Download the file and upload it to the portfolio/ folder on GitHub.
  5. Upload the photos themselves to the portfolio/ folder on GitHub.
  6. Cloudflare redeploys in ~30 seconds — your site updates automatically.

To reorder: rename the number prefixes, re-drop here, regenerate, re-upload manifest.json.
Photos already on GitHub don't need re-uploading unless the filename changed.
